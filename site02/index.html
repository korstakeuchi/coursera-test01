Learning CSS in Coursera

Subheading 1

CSS works by associating rules with HTML elements. These rules govern how the content of specified elements should be displayed.

Element, Class, ID Selector

Elements

p {} - Paragraph Element. This element can only modify the paragraphs in the site.

class Selector

You can create classes that can be used for different elements.

.class00 {} - Creates a class named "class00" with some properties inside it. It can be used by accessing it like this:

<p class= "blue" >

id Selector

Almost the same thing as class Selector but instead of using .name {}, you use #name {}.
And the way to access them is to simply use the id="" attribute.
